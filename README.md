# Python Port Scanner with Banner Grabbing

This project is a simple multi-threaded port scanner built in Python.  
It performs fast port scanning and grabs service banners from open ports.

## Features
- Fast port scanning (1–100 range)
- Multi-threaded scanning
- Banner grabbing (service detection)
- Scan duration tracking
- Report saving (TXT)

## 📌 Usage

```bash
python scanner.py
